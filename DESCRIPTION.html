A tiny plugin to improve Python & Django development experience.
一个用于提升 Python 与 Django 开发体验的 PyCharm 小插件。

功能／Features

  • 生成 __all__ 变量，对变量值列表进行排序，更换引号风格和换行风格等。
    Generate variable __all__ , sort value list, change quotes and line style, etc.
  • 在 reStructuredText 格式的文档字符串中插入超链接。
    Insert hyperlink into reStructuredText format docstring.
  • 插入或覆盖文件头部的 shebang，管理预置的 shebang 。
    Insert or overwrite shebang line on file head, and manage preset shebang list.
  • 通过图形界面创建 Django 应用，同时控制其中文件的创建方式。
    Create a Django app via UI window. And you can create a sub-package instead of any file or skip creation of it.
